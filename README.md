Program Description
	This program reads coordinate data from an input text file and reconstructs it into a hidden message using a 2D array. Each record contains an x-coordinate, a character, and a y-coordinate. The program determines the maximum of both x and y values, creates a grid initialized with spaces, and places character accordingly. Once the grid is full, the program prints the entire array to the console revealing the message.
